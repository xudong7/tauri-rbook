## 如何运行

首先，你需要有以下环境：

1. node & pnpm

2. rust

3. tauri

### 运行

在根目录下运行以下命令：

```bash
# 安装依赖
pnpm install
```

```bash
# 运行
pnpm run tauri dev
```

```bash
# 打包
pnpm run tauri build
```
